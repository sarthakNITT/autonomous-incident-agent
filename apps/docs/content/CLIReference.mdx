# CLI Reference

The `aia` CLI allows you to start the agent, run diagnostics, and manage the system locally.

## Global Options

| Option | Description |
| :--- | :--- |
| `--config <path>` | Path to the configuration file (default: `aia.config.ts`). |
| `--verbose` | Enable verbose logging for debugging. |

## Commands

### `aia start`

Starts all services defined in your configuration. This is the primary command to run the full agent stack.
