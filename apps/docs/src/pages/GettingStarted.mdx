
# Getting Started

Welcome to the Docs App! This guide will help you get started with adding new content.

## Adding Content

To add a new page, simply create a new `.mdx` file in `src/pages`.

### Example

Create a file named `NewPage.mdx`:

```mdx
# My New Page

This is some content.
```

## Running Locally

To run the app locally:

```bash
bun install
bun run dev
```
